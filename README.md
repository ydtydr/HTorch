# HTorch: PyTorch-based Robust Optimization in Hyperbolic Space

# Install
## Requirements
Python >= 3.6
PyTorch >= 1.10.0
CUDA >= 10.1 on linux

## Procedures
1. clone this repo locally
2. run `pip install .`.

We provide examples using HTorch for optimization in hyperbolic space in `examples` folder, please check it out.
